chrome-console

发表于 2018-06-05 更新于 2021-04-14 分类于 JavaScript Valine：

command + shift + j 打开控制台

一般输出

  • console.log() 普通打印
  • console.info() 一般信息
  • console.warn() 警告提示
  • console.error() 错误提示
  • console.debug() 等于 log

大家可能不知道的，console 可以使用 printf 风格的占位符
目前支持字符（%s）、整数（%d或%i）、浮点数（%f）、格式化自身(%c)、格式化 DOM对象(%o) 、 格式化 JavaScript 对象 (%O)。

console.group()

console.groupEnd()

分组打印内容

console.groupCollapsed()

分组打印内容， 默认折叠
