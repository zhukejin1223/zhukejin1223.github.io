数据库管理技术的发展

发表于 2018-03-02 更新于 2021-04-14
人工管理阶段
  1. 数据不保存在计算机中
  2. 数据面向程序
  3. 只有程序的概念，没有文件的概念
  4. 没有专门的软件对数据进行管理
文件系统阶段

优点:

  1. 数据可长期保存在磁盘中
  2. 数据的逻辑结构和物理结构有了区别
  3. 文件组织呈现多样化
  4. 数据不再单独属于某个程序，而是可以重复使用
  5. 对数据的操作以记录为单位

缺点：

  1. 数据之间的联系弱
  2. 数据不一致
  3. 数据冗余
数据库阶段
  1. 采用数据模型来表示复杂的数据结构
  2. 有较高的数据独立型
  3. 数据库系统为用户提供了方便的用户接口
  4. 系统提供数据控制功能
  5. 增加了系统灵活性
高级数据库阶段
  1. 分布式数据库
  2. 面向对象型数据库
  3. Web数据库
数据库发展的三个标志性阶段
  1. 1968年 IBM 推出树状结构的层次模型数据库 IMS
  2. 1969年 CODASYL 提出了网状模型数据库
  3. 1970年 IBM 提出了关系模型数据库
